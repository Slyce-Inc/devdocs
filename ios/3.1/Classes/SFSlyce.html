– openWithAppId:appKey:error:

Initializes a new SFSlyce object using valid App ID and App Key credentials.

- (BOOL)openWithAppId:(NSString *)appId appKey:(NSString *)appKey error:(NSError **)error

Parameters

appId

the App ID.
appKey

the App Key.
error

the pointer to the error object, if any.

Return Value

YES if it succeeded, NO otherwise.

Discussion

Use this method to open the SDK for ‘Public’ recognition functionality.

Warning: As this method performs validation checks of your client ID on our servers, call this method off the main (UI) thread!

Important: Successfully opening the SFSlyce object is mandatory before you start performing any tasks in the SDK!

Important: Successfully opening the SFSlyce object is mandatory before you start performing any tasks in the SDK!
