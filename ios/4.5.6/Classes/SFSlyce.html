– openWithClientID:error:

Initializes a new SFSlyce object using a customer ID.

- (BOOL)openWithClientID:(NSString *)clientID error:(NSError **)error

Parameters

clientID

the customer ID.
error

the pointer to the error object, if any.

Return Value

YES if it succeeded, NO otherwise.

Discussion

Use this method to open the SDK for 2D/3D ‘Premium’ recognition functionality. Depending on your client ID, the SDK will be able to perform Standard and/or Premium tasks.

Warning: As this method performs validation checks of your client ID on our servers, call this method off the main (UI) thread!

Important: Successfully opening the SFSlyce object is mandatory before you start performing any tasks in the SDK!
